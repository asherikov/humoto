9 Doxygen
10 -------
11 
12 - (**dox00**) Doxygen complains about template parameters of classes
13  '@ref humoto::pepper_ik::TaskBaseCoMTracking'
14  and
15  '@ref humoto::pepper_ik::TaskBaseOrientation'.
16  Currently, parameters are altered using 'HUMOTO_DOXYGEN_PROCESSING' macro
17  in order to avoid warnings.
18 
19 - (**dox01**) We use 'typedef' to define tasks ('@ref humoto::TaskALU',
20  '@ref humoto::TaskGIB0', ...), but doxygen does not treat typedefs as
21  classes and does not generate corresponding pages. For this reason, we
22  define fake classes when the source code is processed by doxygen.

26 NAOqi, Aldebaran's Cross Compilation Toolkit
27 --------------------------------------------
28 
29 - (**qi00**) '`cross-config.cmake`' file in the cross compilation toolkit
30  overrides '`CMAKE_CXX_FLAGS`', hence all flags must be passed using
31  '`CMAKE_C_FLAGS`'.
32 
33 - (**qi01**) NAOqi adds wrappers for standard cmake functions to search for
34  dependencies. If these wrappers are not used, e.g., in the bridges, it is
35  necessary to provide hints for cmake's '`find_package`' manually. See
36  '`BOOST_ROOT`' and '`EIGEN3_INCLUDE_DIR`' in CMakeLists.txt
